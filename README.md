# Stock Intelligence Dashboard

A professional AI-powered stock analysis tool built with React + Node.js.

## Features
- Real-time stock data (Yahoo Finance — no API key needed)
- News with AI sentiment tagging (NewsAPI)
- Fear & Greed score calculated from price momentum, RSI, volatility
- Geopolitical events with sector-specific impact analysis
- AI summary generated by Claude

---

## Setup

### Step 1 — Get free API keys

| Service | URL | Free tier |
|---------|-----|-----------|
| NewsAPI | https://newsapi.org/register | 100 req/day |
| Anthropic | https://console.anthropic.com | Pay per use (very cheap) |

### Step 2 — Backend setup

```bash
cd backend
npm install
cp .env.example .env
# Open .env and paste your API keys
npm run dev
```

Backend runs on: http://localhost:5000

### Step 3 — Frontend setup (new terminal)

```bash
cd frontend
npm install
npm start
```

Frontend runs on: http://localhost:3000
